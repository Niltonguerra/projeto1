## tipos de analise de uma ação:
### **Análise Fundamentalista** 
Avalia o valor intrínseco da empresa via dados financeiros (lucro, patrimônio, caixa, dívida). Métodos: múltiplos (P/L, EV/EBITDA, P/VP) e valuation absoluto (DCF, Dividend Discount Model). Fonte de dados: CVM (dados contábeis/cadastrais), Brapi (preço para comparar com valor calculado).
aprofundamento:[[aprofundamento analise Fundamentalista]]

### **Análise Técnica** 
Avalia padrões de preço e volume histórico para prever movimento futuro, sem olhar fundamentos. Indicadores: médias móveis, RSI, MACD, suporte/resistência. Fonte de dados: Brapi (série histórica de preço/volume).
aprofundamento:[[aprofundamento Analise Técnica]]

### **Análise Quantitativa/Fatorial**
Rankeia ações por exposição a fatores estatísticos comprovados (value, momentum, quality, size, low-volatility). Não avalia uma ação isolada, mas dentro de um universo comparável. Fonte de dados: Brapi (preço/retorno histórico) + CVM (dados fundamentalistas para compor fatores value/quality).
aprofundamento:[[aprofundamento Analise Quantitativa-Fatorial]]

### **Análise Macro/Setorial (top-down)**
Não avalia a ação diretamente — ajusta parâmetros das outras análises (taxa de desconto, múltiplo aceitável, alocação setorial) com base em cenário macroeconômico. Fonte de dados: BC (Selic, IPCA, câmbio), IBGE (indicadores de atividade econômica real).
aprofundamento:[[aprofundamento Analise Macro-Setorial_top-down]]
