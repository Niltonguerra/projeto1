## tipos de analise de uma ação:
### **Análise Fundamentalista** 
existe, é a mais consolidada, tem literatura acadêmica robusta (Graham, Damodaran) e é praticada por value investors e a maioria dos analistas sell-side.


### **Análise Técnica** 
existe, é praticada, mas tem menos respaldo acadêmico (hipótese de mercados eficientes questiona sua validade estatística de forma consistente).


### **Análise Quantitativa/Fatorial**
é dominante na gestão institucional moderna (quant funds), mas é mais um framework de portfólio do que de avaliação individual de ação.


### **Análise Macro/Setorial (top-down)**
existe como prática, mas funciona como camada de contexto/filtro, não como método autônomo de avaliação de uma ação isolada.
